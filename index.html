My research: Topic Models (Latent Dirichlet Allocation)

In a nut shell, statistical Topic Modelling is the ML technique for uncovering the topics that pervade a text corpus.

Background on Probabilistic Topic Models. The basic idea behind TMs is that, given a large unstructured text, the hidden topics are discovered. TMs are directed to answer the question: “what is this text talking about?”. This concept is illustrated in Figure 1 using a toy example (as the text analyzed may contain billions of words). Here, each of the words from the text is assigned to a topic, i.e., a list of related words on the same subject. The output of a TM is the most likely words per each of the most likely topics. We will see later that the number of topics and the number of words are parameterizable. A more formal definition of TMs, is that they are a set of machine learning techniques aimed to extract the thematic structure in a massive collection of documents. These are Information Recovery (IR) / Data Mining (DM) techniques that use the texts latent information for clustering the documents; i.e., grouping them according to the similarity found amongst them.
